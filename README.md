# Django To-Do App

A To-Do application.

## Features
- Authentication 
- Add tasks
- Edit tasks
- Delete tasks

## Tech Stack

- Python
- Django

## Run the Project

```bash
git clone <repository-url>
cd todo_project
python -m venv venv
source venv/bin/activate   # Linux/Kali

pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

Open in browser:
```
http://127.0.0.1:8000
```
